This document outlines the core design principles and token values for our product's user interface.

## **Theme Overview**

Our current theme emphasizes a **light** aesthetic, providing a modern and sophisticated user experience.

## **Colors**

The brand's primary color is a striking blue, supported by a refreshing teal and an energetic amber.

*   **Primary Color:** `#3F51B5` (A deep, reliable blue)
*   **Secondary Color:** `#009688` (A calming, vibrant teal)
*   **Tertiary Color:** `#FFB300` (A warm, engaging amber)
*   **Neutral Color:** (Not specified, derived from the light theme)

<h2>**Typography**</h2>

The typography is designed for clarity and modern appeal.

*   **Headline Font:** Plus Jakarta Sans
*   **Body Font:** Inter
*   **Label Font:** Inter

<h2>**Shape and Spacing**</h2>

*   **Roundedness:** UI elements feature moderate roundedness (level 2), striking a balance between modern sharpness and approachability.
*   **Spacing:** A normal level of spacing (level 2) ensures a balanced and comfortable visual density.
